基于面向对象的绘图工具设计与实现 
主要功能要求: 
1) 提供绘制直线、矩形、椭圆、三角形、五角星等简单图形元素,每个图形元
素包括大小、位置、线宽、前景颜、填充颜色(直线没有填充颜色)等属性。
通过工具条和下拉菜单方式驱动绘制(类似与画笔工具)。 
2) 对每个图形对象可以在图形显示区域和列表显示区域选中,选中后可以对
其进行编辑,包括改变属性、删除、改变绘制顺序等,也可以实现对象复制。 
3) 对所绘制的图形对象分别以图形方式和列表方式显示,其中在列表方式显
示框中可以根据图形对象的名称、前景颜色和笔宽进行排序。 
4)所绘制的简单图形对象可以组合成组合对象。 
5) 所绘制的图形可以保存到文件中,下次打开可以继续进行绘制和修改。 
设计与实现要求: 
1)采用面向对象思想进行设计与实现; 
2)在设计过程中引入工厂模式、组合模式、原型模式等设计模式。


使用QT6进行的编写，Build文件也以放进去。
